# 異國情調花園
黏液科技的附加 繁體翻譯版<br>

異國情調花園是一個大型附加,為遊戲帶來了各種新的植物，水果，蔬菜和食物.<br>

[官方原碼連結](https://github.com/TheBusyBiscuit/ExoticGarden)
